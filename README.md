# Voclip
Веб сервис для изучения анлгийского языка через личный удобный словарь и контент из youtube.
Основная идея сервиса заключается в том, что пользователь формирует свою базу слов и выражений во время изучения языка, а затем закрепляет их с помощью встроенной тренировки.

Сервис состоит из трех основных частей:
 - персональный словарь
 - пространство с youtube-видео
 - тренировка для закрепления изченного материала

### Персональный словарь
Пользователь может создавать собственную базу английских слов и выражений для дальнейшего изучения.
Заметка может содержать:
- отдельные слова
- фразовые глаголы
- устойчивые выражения
- языковые конструкции

Для каждой заметки можно сохранить:
- один или несколько переводов
- один или несколько примеров
- источник изучения

Источником может быть:
- контекст из youtube-видео

### Изучение через Youtube
Сервис предоставляет пространство с коротким youtube-видео для изучения языка в реальном контексте.
Во время просмотра пользователь может добавить новые слова и выражения в свой словарь. Если слово уже присутствует в словаре пользователя, оно дополнительно подсвечивается в контексте.
Таким образом пользователь не изучает слово изолированно, а встречает их  в релаьной речи и контексте.

### Тренировка
После добавления слов в словарь пользователь может закрепить их с помомщтю тренировки.
Для каждой заметки рассчитывается рейтинг изучения:
- при правильном ответе рейтинг увеличивается
- при неправильном ответе рейтинг уменьшается

Чем ниже рейтинг заметки, тем чаще слово будет появляться в тренировке

### Прогресс Изучения
Сервис отображает небольшой блок изученных слов в день, а также сколько слов добавлено в словарь за день

### Авторизация
Весь функционал доступен только авторизованным пользователям. Регистрация и вход выполняются через подтверждение эдектронной почты. Пользователь указывает email-адрес и получает одноразовый код для входа в систему.
