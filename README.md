# Voclip
Веб сервис для изучения анлгийского языка через личный удобный словарь с заметками слов или сложных фраз с контекстной тренировкой.
Пользователь на этом сервисе может создавать свою базу слов и выражений, как из просмотренного видео, так и в самом словаре. Пользователь закрепляет слова и улучшает свой уровень английского.
Причем заметка для иузчения может содержать: 
 -  простые слова
 -  фразовые глаголы
 -  устойчивые выражения
 -  целые конструкции
Для каждой заметки можно сохранить:
 -  перевод(ы)
 -  пример(ы)
 -  источник изучения с примеров использования этого слово
В каждой заметки есть источник (откуда взято слово):
 - собственные примеры пользователя
 - примеры сгенерированные языковой моделью
 - контекст из youtube-видео
У каждого слова есть свой рейтнг обучения. Когда пользователь в тренировке неправильно выбирает вариант ответа, то  рейтинг понижается, если правильно, то  рейтинг увеличивается. Чем ниже рейтинг слова, тем чаще это слово будет встречаться пользователю для закрепления.
В словаре также есть  блок прогресса, чтобы отслеживать сколько  пользователь изучил за день слов.

На серивисе есть  короткие youtube-видео по которым пользователь может собирать свой словарь. Когда пользователь смотрит видео он видит подсвеченные слова которые уже есть в его словаре.
Весь контент на серивисе доступен пользователям, которые прошли авторизацию/регистрацию. Авторизация/регистрация реализована через ввод кода, который приходит на почту, которую указал пользователь.
